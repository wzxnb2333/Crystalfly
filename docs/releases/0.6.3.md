# Crystalfly 0.6.3

## 功能完整性复核

- 对最新 `main` 与保留旧功能的本地开发工作区执行声明、属性、命令绑定、样式类、本地化键、资源和测试用例差分审计。
- 配置编辑、实例存档编辑、骑士加载动画、依赖树、顶部设置 Tab、“选择实例”、页面淡入和后台版本扫描均已确认存在。
- 存档编辑命令已升级为异步实现；大型字段列表继续使用真实虚拟化，不属于功能缺失。

## 窗口与持续集成

- 恢复原生可缩放窗口边框，同时继续使用自绘标题栏和最小化、最大化、关闭按钮。
- Windows 11 启动时明确请求 DWM 系统圆角；Windows 10 对不支持的属性保持兼容。
- 恢复 Windows PR 构建测试工作流，并在贡献指南中记录按模块运行测试的方法。

本版本不修改设置 Schema、实例目录、LocalLow、快照、下载队列或缓存格式。
