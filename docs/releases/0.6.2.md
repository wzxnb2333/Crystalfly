# Crystalfly 0.6.2

## 导航与页面反馈

- 设置入口恢复为顶部主导航文字 Tab，不再隐藏在窗口控制区的齿轮按钮中。
- 启动页入口和实例列表标题统一为“选择实例”。
- 主页面、实例详情子页和 Mod 批量操作栏恢复 180ms 透明度淡入。
- 页面动画只改变透明度，并在 Windows 关闭客户端动画或启用减少动态效果时停用。

## 后台扫描与启动响应

- 启动后立即使用内嵌目录扫描版本根目录，不再等待 GitHub 远程目录刷新。
- Steam 自动重连与实例扫描并行，不再阻塞版本文件识别。
- 游戏构建识别、Loader 检查和 Mod 状态读取移到后台线程；多个实例并行检查。
- 远程目录刷新完成后安全执行二次扫描，避免初次扫描与目录更新竞争。
- 后台目录解析失败只记录日志，不会在关闭程序时重新抛出导致残留进程。

本版本不修改设置 Schema、实例目录、LocalLow、快照、下载队列或缓存格式。
