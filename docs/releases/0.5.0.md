# Crystalfly 0.5.0

## 主要变化

- 实例详情新增 Mod 预设：创建、重新捕获、复制、删除、导入、导出、追加/精确应用和应用前状态恢复。
- 预设绑定精确游戏构建、Loader 与 Mod 版本；本地或外部 Mod 仅记录名称和 SHA-256。
- 预设应用复用下载队列，按依赖顺序安装、启用和停用；同一实例的预设组串行执行。
- 精确模式保留固定 Mod 及其传递依赖，恢复时会卸载应用前不存在的 Mod。
- 新增 12 位分享码服务与 `crystalfly://modpack?code=` 分享页；本地 JSON 导入导出不依赖网络。

## 数据兼容

现有设置、实例、LocalLow、存档快照、下载队列和缓存位置保持不变。预设保存在 `<版本根目录>\.crystalfly\instances\<实例 ID>\presets`。应用前恢复点仅包含受影响 Mod 的安装与启停状态。
