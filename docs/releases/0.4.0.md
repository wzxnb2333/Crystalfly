# Crystalfly 0.4.0

## 主要变化

- Mod 市场加入由官方 ModLinks 历史生成的“最近新增 / 最近更新”筛选。
- Mod 详情按需加载经过清理的 README 与最新发行说明，并使用 ETag 原子缓存。
- 官方受管理 Mod 支持保持启停状态的重装/修复、安装目录、日志和实例隔离全局设置入口。
- 设置页支持绑定精确游戏构建与 Modding API Loader 的 HTTPS 自定义 ModLinks 完整替换源。
- 自定义 XML 禁止 DTD，Windows 包继续强制 HTTPS 与 SHA-256；自定义源始终未验证。

## 数据兼容

现有设置、实例、LocalLow、快照、下载队列和包缓存保持原位置。新增活动目录和 Mod 内容缓存位于 `%LOCALAPPDATA%\Crystalfly\catalog` 与 `%LOCALAPPDATA%\Crystalfly\content`；便携模式对应程序旁 `Data`。
