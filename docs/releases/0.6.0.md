# Crystalfly 0.6.0

## 外部命令

- 注册并解析受限的 `crystalfly://` 命令集。
- 第二进程通过当前用户命名管道把链接转发到已运行窗口。
- 所有修改状态的外部命令先展示解析后的操作摘要；外部链接不能请求强制启动或注入包 URL、本地路径。

## 客户端更新

- 每 24 小时检查一次稳定通道，支持关闭自动检查、手动检查、稍后处理和跳过指定版本。
- 更新清单使用 Ed25519 签名；客户端同时验证稳定通道、Windows x64、版本、资产大小和 SHA-256。
- 更新器在主程序退出后再次验证并锁定资产，直到安装或替换完成，避免已校验文件在提权前被替换。
- GitHub 直连/镜像和全局离线策略同时作用于更新清单与资产下载。
- 安装模式调用静默 Inno 安装包；便携模式在目标同卷暂存和备份，使用持久恢复日志并等待新程序健康握手后清理备份，失败或中断时可恢复原程序并始终保留 `Data`。
- 更新资产在更新器得到确定结果后清理；启动时会删除过期孤儿资产。

## 发布

- 安装包注册 `crystalfly://` 协议并在卸载时移除。
- Release 包含便携 ZIP、安装包、签名 `update-manifest.v1.json` 和 `SHA256SUMS.txt`。
